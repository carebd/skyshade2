## Quick Start вАФ Shared Hosting (cPanel)

### Step 1 вАФ Upload files

Upload all project files to your hosting directory (e.g. `public_html` or a subfolder like `public_html/drive`).

### Step 2 вАФ Create a MySQL database in cPanel

1. Log in to **cPanel**
2. Go to **MySQL¬Ѓ Databases**
3. Create a new database (e.g. `cloudrive`)
4. Create a new database user with a strong password
5. **Add the user to the database** and grant **ALL PRIVILEGES**

> вЪ†пЄП cPanel always adds your account username as a prefix.  
> If your cPanel username is `mysite`, the database will be `mysite_cloudrive` and the user will be `mysite_cduser`.  
> Use these full names in the installer.

### Step 3 вАФ Run the web installer

Open your browser and visit:

```
https://yourdomain.com/install.php
```

Fill in:

| Field | Value |
|---|---|
| DB Host | `localhost` (almost always) |
| Database Name | Full name with prefix, e.g. `mysite_cloudrive` |
| DB Username | Full name with prefix, e.g. `mysite_cduser` |
| DB Password | The MySQL user password you set in cPanel |
| Site URL | `https://yourdomain.com` (no trailing slash) |
| Admin Username | `Admin` |
| Admin Password | Your chosen password (min 8 characters) |
| Admin Email | Your email address |

Click **рЯЪА Install CloudDrive Now**.

### Step 4 вАФ Log in

Visit `https://yourdomain.com/login.php` and sign in with:

- **Username:** `Admin`  
- **Password:** *(the password you entered in the installer)*

## Security Notes

- `config.php` is listed in `.gitignore` and protected by `.htaccess` вАФ never share it.
- The `storage/` directory is blocked from direct web access via `.htaccess` and a `Deny from all` rule inside the folder.
- All file downloads go through `download.php`, which verifies ownership or a valid share token.
- Passwords are hashed with bcrypt (cost 12).
- Sessions expire after 1 hour of inactivity (configurable via `SESSION_LIFETIME` in `config.php`).
